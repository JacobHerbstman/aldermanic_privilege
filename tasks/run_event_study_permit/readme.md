# Permit event study

Runs the paper-facing 2015 permit event study. The main specification uses permits indexed by application date, the stringency score estimated through 2014, a 500ft boundary sample, pre-period permit controls, and ward-pair-clustered standard errors.

Produces:
- `output/event_study_permit_2015_high_discretion_application_frozen2014_itt_preperiod_controls_500ft_clust_ward_pair.pdf`
- `output/event_study_permit_2015_low_discretion_nosigns_application_frozen2014_itt_preperiod_controls_500ft_clust_ward_pair.pdf`
- `output/event_study_permit_2015_high_discretion_application_frozen2014_stable_preperiod_controls_500ft_clust_ward_pair.pdf`
- `output/did_table_permit_2015_application_frozen2014_itt_preperiod_controls_500ft_clust_ward_pair.tex`

Approx. runtime: ~3-20 minutes.
