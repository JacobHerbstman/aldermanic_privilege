# Historical density coordinates

This task contains the frozen original-PIN coordinates used when a construction
record has no coordinates in the 2025 Parcel Universe. Coordinates come from
the construction-year Historical Parcel Universe or, for four records, from a
historical property-address match. The file is committed so the paper build
does not depend on later revisions to Cook County's live data services. Its
full derivation is retained in `tasks/audits/density_project_lineage`.
