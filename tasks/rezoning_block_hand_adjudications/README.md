# Rezoning Block Hand Adjudications

This task publishes 148 reviewed matter-to-block links for nine ordinances whose
geography cannot be represented by an ordinary address point. The decisions use
City zoning polygons linked by Clerk document, ordinance, or planned-development
number and four explicit legal-description or parcel reviews.

The Makefile intentionally only verifies that the committed output is present.
The reconstruction audit is in `tasks/audits/rezoning_block_assignment_review/`.
