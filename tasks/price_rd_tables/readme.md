# price_rd_tables

Purpose: Builds the paper-facing price RD controls table from the listed-rent and home-sales RD panels.

Produces: `output/price_rd_controls_panels_bw500.tex`, a two-panel table with listed rents in Panel A and home sales in Panel B. Both panels report the same control waterfall rows and cluster standard errors at the boundary-segment level.

Approx. runtime: <1 minute once upstream rent and sales RD outputs exist.
