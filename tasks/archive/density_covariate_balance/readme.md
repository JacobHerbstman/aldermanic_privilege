# density_covariate_balance

Purpose: Tests whether the ward-level control variables used in the main `500 ft` multifamily density FE specification are themselves discontinuous with the border-pair stringency gradient.

Produces: `output/density_covariate_balance_main_bw500_multifamily.csv`, `output/density_covariate_balance_main_bw500_multifamily.tex`, `output/density_covariate_balance_snapshot2014_bw500_multifamily.csv`.

Approx. runtime: ~1-5 minutes.
