This task benchmarks the citywide, ward, and community-area RentHub aggregates against ACS rent growth.

The main benchmark window is 2014-2019. Ward growth comparisons use 2015-2019 because the ward map changes between 2014 and 2015, while the citywide and community-area comparisons stay on the full 2014-2019 pre-COVID window.

The preferred private series is `renthub_clean`. `renthub_raw` is carried through as a sensitivity series in the CSV diagnostics.
