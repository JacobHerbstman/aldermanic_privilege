This task benchmarks the citywide, ward, and community-area private home-sales aggregates against ACS owner-occupied home values.

The main benchmark window is 2014-2019. Ward growth comparisons use 2015-2019 because the ward map changes between 2014 and 2015, while the citywide and community-area comparisons stay on the full 2014-2019 pre-COVID window.

The point of the comparison is growth comparability, not equality in levels, because the private series is a median sale price while the ACS object is a stock-value measure.
