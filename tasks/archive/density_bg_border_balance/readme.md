# density_bg_border_balance

Purpose: Checks parcel-level block-group covariate balance for the main `500 ft` multifamily border design using a fixed 2014 ACS snapshot, and reports how much of the sample comes from block groups that straddle both sides of the same ward-pair segment.

Produces: `output/density_bg_border_balance_snapshot2014.csv`, `output/density_bg_border_balance_snapshot2014_drop_straddle.csv`, `output/density_bg_border_straddle_summary_snapshot2014.csv`, `output/density_bg_border_balance_snapshot2014.tex`.

Approx. runtime: ~1-5 minutes.
