# density_amenity_balance

Purpose: Checks whether exact parcel amenities and zoned FAR line up with the stringency gradient in the main 500-foot multifamily density design.

Produces: Produces files such as `output/density_amenity_balance_bw500_multifamily.csv`, `output/density_amenity_balance_bw500_multifamily.tex`, and related task-specific outputs in `output/`.

Approximate runtime: 2-8 minutes.
