# Density covariate balance

This task produces the appendix table comparing observed neighborhood
characteristics across the ward boundaries used in the density analysis. The
table uses the paper's binary boundary comparison and reports adjusted
differences in standard deviation units for all construction and multifamily
construction.

The joint tests allow the balance equations to be correlated within ward
pairs. Broader balance diagnostics and alternative spatial covariance
estimators remain in `tasks/audits/density_covariate_balance`.
