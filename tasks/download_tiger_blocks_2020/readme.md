# Download TIGER Blocks 2020

Downloads the U.S. Census Bureau Illinois 2020 Census block TIGER shapefile from the 2025 TIGER release:

`https://www2.census.gov/geo/tiger/TIGER2025/TABBLOCK20/tl_2025_17_tabblock20.zip`

The downstream `census_block_2020_cleaning` task clips this statewide file to Chicago.
