\subsection{Event Study Robustness: Yearly Specification}

This subsection presents robustness checks for the yearly event study results in Section~\ref{sec:empirical_results_rent_home_vals}. I consider two alternative specifications: (1) weighting observations by the number of transactions in each census block-year cell, and (2) including time-varying demographic controls for median household income, racial composition, and homeownership rates.

\subsubsection{Home Sale Prices: Weighted Specification (Yearly)}

Figure~\ref{fig:event_study_sales_weighted} presents the event study results for home sale prices using observation weights proportional to the number of sales in each census block-year cell. The results are qualitatively similar to the unweighted specification in the main text.
